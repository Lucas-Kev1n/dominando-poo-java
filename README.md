# 🚀 Dominando a Programação Orientada a Objetos & Java Avançado

Este repositório foi criado para centralizar, organizar e documentar a minha jornada de aprendizado nos pilares essenciais da Programação Orientada a Objetos (POO) e nos recursos avançados do ecossistema Java.

O objetivo principal é construir uma base técnica sólida e alinhada com as melhores práticas de mercado, preparando o terreno para o desenvolvimento de APIs robustas com Spring Boot.

---

## 📁 Estrutura do Repositório e Cronograma de Módulos

O projeto está estruturado em pastas modulares. Cada diretório contém problemas do mundo real resolvidos aplicando conceitos arquiteturais específicos:

### 🟢 01. Construtores, Sobrecarga e Encapsulamento (`/01-encapsulamento-conta`)
* **Regra de Negócio:** Sistema de cadastro e movimentação de contas bancárias com taxas de saque operacionais.
* **Conceitos Aplicados:**
    - **Encapsulamento Rígido:** Modificadores `private` protegendo o estado do objeto e controle de acesso via Getters e Setters estratégicos.
    - **Sobrecarga de Construtores:** Flexibilidade na instanciação de objetos (abertura de conta com ou sem depósito inicial).
    - **Métodos de Negócio:** Lógica interna mapeada diretamente na entidade para manipulação segura de saldo.

### 🟡 02. Enumerações e Composição
* **Foco do Aprendizado:** Modelagem de sistemas complexos dividindo responsabilidades entre múltiplos objetos (relação "tem um") e definição de estados constantes em Enums.

### 🔴 03. Herança e Polimorfismo
* **Foco do Aprendizado:** Reaproveitamento de código extensível, sobrescrita de métodos (`@Override`), uso de classes/métodos abstratos e comportamento dinâmico em tempo de execução.

### 🔵 04. Tratamento de Exceções
* **Foco do Aprendizado:** Construção de sistemas resilientes e seguros utilizando blocos `try-catch` e criação de exceções personalizadas de negócio.

### 🟤 05. Interfaces e Desacoplamento
* **Foco do Aprendizado:** Inversão de controle e criação de contratos para garantir o baixo acoplamento entre os componentes do software (preparação direta para a arquitetura do Spring).

### 🟣 06. Tópicos Avançados (Generics, Set, Map, Lambda e Stream API)
* **Foco do Aprendizado:** Alta performance em manipulação de coleções de dados, tipagem dinâmica segura e uso de Programação Funcional para escrita de código limpo e moderno.
